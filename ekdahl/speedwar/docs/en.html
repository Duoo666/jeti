Speed warnigns lua

Helps you fly at safe speed, without having a lot of false alarms while on ground or doing repeated takeoff and landings. Warnings with audio and haptic.

Supporting the following warnings:

  • OverSpeed - plays audio file when you fly faster than this speed
  • MinimumSafeSpeed - plays audio file when you fly slower than minimum safe speed. This alarm is enabled after the first time you hit the safe speed, so it will not sound on the ground.
  • TakeOffSpeed - plays audio file when takeoff speed is ok and ready to rotate and will not alarm in the air. Will reset if you are under takeoff speed for 10 seconds. Will not play on the ground.
  • StallSpeed - plays audio file when stall speed limit is reached - This alarm is enabled after the first time you hit the stall speed, so it will not sound on the ground.
